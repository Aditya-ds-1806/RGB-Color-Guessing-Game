How to Play?

The RGB color model is an additive color model in which red, green and blue light are added together in various ways to reproduce a broad array of colors (16 million colours). The name of the model comes from the initials of the three additive primary colors, red, green, and blue.

Every time you click on New Game, a new RGB colour code is randomly generated and displayed just above the navbar. Every RGB code is assosciated with 3 numbers that range between 0 and 255. Depending on the level of difficulty you choose, you will also see 3/6 colours on the page. Only one among these 3/6 colours correspond to the RGB colour code displayed on the page. Your task is to correctly identify the colour that corresponds to the displayed RGB colour code. You also have 3 lives - make the most out of it.
